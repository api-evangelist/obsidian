Improvements Obsidian CLI Added autocompletion for Obsidian commands to the terminal interface (TUI) when using the id= parameter. Bug fixes Fixed several issues with copy, cut, and paste when no text is selected. Copying and pasting a line no longer places the cursor in the wrong position. In tables, copying or cutting without a selection now correctly copies only the cell contents. Pasting a URL while text is selected across multiple cursors now wraps each selection as a Markdown link ( [selected text](url) ).
